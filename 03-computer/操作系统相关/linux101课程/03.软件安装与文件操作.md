# 软件安装与文件操作

本章目标：
- 通过命令行安装所需要的软件。
- 对文件与目录进行创建、移动、删除等。
- 压缩、解压文件
- 对没有使用过的软件能够更快的了解使用方法

## 使用应用商店安装

这里直接使用自带的应用商店的发行版安装即可。

## 使用包管理器系统安装

包管理器系统有很多：
常见比如`debian`(.deb)软件包的`dpkg`以及他的前端的`apt`；
`rpm`包管理器以及它的前端`dnf`、前端`yum`;`pacman`包管理器等等。

本文使用`apt`作为典型实例进行讲解。

搜索：

在搜索包前，可以使用`apt search`命令搜索软件仓库，查看对应的包名是否在仓库中。

    apt search firefox

安装：

在确定软件包的包名之后，可以通过`apt install <包名>`安装软件包。如果一次需要安装多个包，可以使用`apt install <包名1><包名2>···`的写法进行安装。

## 官方软件源镜像：

通过`apt`安装的软件都来自于相对应的软件源，每个`linux`发行版一般都带有官方的软件源，在官方的软件源中已经包含了丰富的软件，apt的软件源列表在`/etc/apt/sources.list`下。
